A Mustache Sinatra Example
==========================

Get started:

    $ gem install mustache
    $ git clone git://github.com/defunkt/mustache-sinatra-example.git
    $ cd mustache-sinatra-example
    $ rackup config.ru
    $ open http://localhost:9292

Or browser to <http://localhost:9292> by hand.

Enjoy.
